# Django REST Framework + Docker + PostgreSQL second project for my skills

Backend-приложение на Django REST Framework, развернутое с помощью Docker и Docker Compose, с базой данных PostgreSQL и настройками через .env.

---

## Стек технологий
- Python 3.11
- Django
- Django REST Framework
- PostgreSQL
- Docker
- Docker Compose
